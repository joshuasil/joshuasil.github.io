# Publications

### Peer-Reviewed Publications

**An Artificially Intelligent, Natural Language Processing Chatbot Designed to Promote COVID-19 Vaccination: A Proof-of-concept Pilot Study**  
Zhou, S., **Silvasstar, J.**, Clark, C., Salyers, A., Chavez, C., Bull, S. (2023)  
*Digital Health* • [DOI: 10.1177/20552076231155679](https://doi.org/10.1177/20552076231155679)

Built and deployed a text message-based chatbot using Flask, Twilio, and custom ML classification algorithms. Engineered the machine learning pipeline (SVM, Naive Bayes, LSTM) to optimize response accuracy for vaccine-related questions across multiple FQHCs.

---

**Developing Record Linkage Methods to Improve Data Quality**  
Ong, T.C., Voong, C., Kwan, B., et al. (2023)  
*Patient-Centered Outcomes Research Institute (PCORI)* • [DOI: 10.25302/04.2023.ME.2018C111287](https://doi.org/10.25302/04.2023.ME.2018C111287)

Developed Python automation scripts for deterministic and probabilistic record linkage across multiple health datasets. Created data quality metrics and deployed solutions used by two partner institutions.

**Efficacy of an mHealth Intervention (BRAVE) to Promote Mental Wellness for American Indian and Alaska Native Teenagers and Young Adults: Randomized Controlled Trial**  
Craig Rushing, S., Kelley, A., Bull, S., et al. (2021)  
*JMIR Mental Health* • [DOI: 10.2196/26158](https://doi.org/10.2196/26158)

Managed randomization for 1,000+ participants and conducted statistical analysis in R. Built Tableau dashboards to visualize resilience and self-esteem outcomes across intervention arms.
